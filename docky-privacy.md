# Docky Privacy Policy

Last updated: 2026-05-03

Contact: ludeveloper97@gmail.com

Docky is a local-first Android launcher. It does not include ads, analytics,
tracking SDKs, login accounts or third-party data sales.

## Data Used On Device

Docky reads the list of launchable apps installed on the device only to display,
search, group and launch local apps. This app inventory is processed on device
and is not uploaded by Docky.

Docky stores launcher settings, selected icon pack, home slots, groups, profile
display data, friend display data, recent play activity and user-selected images
locally on the device.

Album images and custom images are selected by the user through the system
picker and copied into Docky app storage. Docky does not scan the full device
gallery and does not request broad photo access.

## Backups

Backups are created only when the user exports them. A backup file can contain
launcher settings, app package names, profile names, groups, recent activity and
user-selected images. Docky does not upload backup files. Exported backup files
are controlled by the user.

## Data Sharing

Docky does not sell, share or transmit personal data to third parties.

## Data Deletion

Users can delete local Docky data by clearing the app storage in Android system
settings or uninstalling the app. Exported backup files are controlled by the
user and can be deleted manually.

## Changes

This policy may be updated when Docky changes its data handling. The updated
policy should be published before submitting a new Play Store release.
